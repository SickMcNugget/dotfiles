# github.com/SickMcNugget/dotfiles

SickMcNugget's dotfiles, managed with [`chezmoi`](https://github.com/twpayne/chezmoi).

Install them with:

```console
chezmoi init SickMcNugget
```

If you're me, and you're developing the dotfiles, install them with:

```console
./install.sh
```

Personal secrets are stored in [Proton Pass](https://proton.me/pass) and you'll
need the [Proton Pass CLI](https://protonpass.github.io/pass-cli) installed.
Login to Proton Pass CLI with:

```console
pass-cli login
```
